## ✨ Key Features

### 1. 동아리 소개 (Club Introduction)
- 분과별 동아리 슬라이드
- 클릭 시 상세 동아리 정보 제공

### 2. 동아리 추천 페이지 (Club Recommendation)
- 맞춤형 설문을 통한 동아리 추천 시스템

### 3. 동아리방 찾기 (Club Room Finder)
- WebGL 기반 인터랙티브 캠퍼스 맵
- 동아리방 위치 및 정보 제공

### 4. 메인 페이지 (Main Page)
- 다양한 애니메이션
- 스크롤 이벤트 및 인터랙티브 디자인
